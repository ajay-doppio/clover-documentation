Control_Loop_FT_CheckItemType

Overview

This boolean funtion returns true if the value of iITTY(Item Type) collected in the last API call (MMS200MI_Get) is not equals to "T80". But if the iITTY value is equal to "T80", then cat.info() statements are printed and the control comes out of the loop.

Input Variables

  • iITTY - ITTY

Code


          iITTY = iITTY == null ? "" : iITTY.trim();

		
		return !iITTY.equals("T80");
