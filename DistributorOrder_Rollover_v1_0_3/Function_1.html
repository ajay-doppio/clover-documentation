Function_1

Overview

This fucntion creates a variable from the values of customer number(iCUNO_Old), customer name(iCUNM) and order number(iORNO_Old) collected from the previous API calls in a string format separated by colons and assigns it to a variable oPARM. This oPARM is sent as batch text value in order details in the next API call.

Input Variables

  • iCUNO_Old - CUNO_Old
  • iCUNM - CUNM
  • iORNO_Old - ORNO_Old

Code


          iCUNO_Old = iCUNO_Old == null ? "" : iCUNO_Old.trim();

		iCUNM = iCUNM == null ? "" : iCUNM.trim();

		iORNO_Old = iORNO_Old == null ? "" : iORNO_Old.trim();


		oPARM = iCUNO_Old + " : " + iCUNM + " : " + iORNO_Old;

Output Variables

  • oPARM - PARM
